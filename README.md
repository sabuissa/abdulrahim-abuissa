# Abdulrahim Abuissa — Personal Website

A personal website for Abdulrahim Abuissa: Qatari entrepreneur, investor, professional basketball player, and FIBA Foundation Youth Leader.

## Client
Abdulrahim (my brother) is the first Qatari to play NCAA Division I basketball and founder of Abuissa Sports.

## About the Site
A three-page personal site (Home, About, Contact) introducing his story across basketball, entrepreneurship, and community work, with a contact form and social links. Built with semantic HTML, an external CSS stylesheet, a responsive Flexbox/Grid layout, and a charcoal-and-gold theme.

## Live Site
https://sabuissa.github.io/abdulrahim-abuissa

## What I Learned
- How to structure a multi-page site with shared external CSS and consistent navigation
- Semantic HTML (header, nav, main, section, footer) and why it matters
- Responsive layout with Flexbox and Grid, and using background images with overlays
- That filenames are case-sensitive on GitHub Pages, and why committing/pushing is separate from saving
- How gathering real client requirements shapes design decisions and how a v1 can be scoped as an iteration toward a larger vision
